# 面向对象设计的五大原则 **SOLID**

-   S - Single Responsibility Principle - 单一职责原则
-   O - Open Close Principle - 开闭原则
-   L - Liskov Principle of Substitution - 里式替换原则
-   I - Interface Segregation Principle - 接口隔离原则
-   D - Dependency Inversion Principle - 依赖倒置原则
